# Rule-Based AI Chatbot

## Project Overview
This project is a Rule-Based AI Chatbot developed using Python as part of the DecodeLabs Artificial Intelligence Internship.

## Features
- Handles greetings
- Answers predefined questions
- Uses dictionary-based responses
- Continuous conversation loop
- Exit commands support
- Fallback response for unknown inputs

## Technologies Used
- Python 3

## How to Run
1. Download chatbot.py
2. Open Terminal/Command Prompt
3. Run:

python chatbot.py

## Inputs
- hi
- hello
- how are you
- what is your name
- help
- bye
